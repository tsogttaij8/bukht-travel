# BUKHT Launch Roadmap

## Phase 1: Core Operations

- Replace JSON storage with a real persistent database layer
- Make cargo tracking use live shipment data and status history
- Build an admin dashboard for shipment creation and status updates
- Keep email-code login, but move its data into the database

## Phase 2: Commercial Readiness

- Add product management for `shop`
- Add eSIM package management and order intake
- Add travel inquiry and booking intake
- Add payment confirmation flows

## Phase 3: Launch Hardening

- Add rate limits for auth endpoints
- Add privacy policy, terms, and contact pages
- Add analytics, SEO metadata, and monitoring
- Add deployment checklist and backup process
